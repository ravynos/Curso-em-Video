
## Soma

## Subtração

```python

```

## Multiplicação

```python
5 * 2 == 10
```

## Divisão

```python

```

## Exponenciação

```python

```

## Divisão inteira


```python

```

# Ordem de precedência

## 1° 
- Parênteses ()
## 2°
- Exponenciação **
## 3° 
- Multiplicação *
- Divisão /
- Divisão Inteira //
- Modulo %
## 4° 
- Soma +
- Subtração -

OBS.: Caso apresente o mesmo operador de ordem igual, deve ser considerado sempre o que aparece primeiro.
